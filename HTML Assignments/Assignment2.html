We will see all the text formatting tags using a Table

HTML Formatting Tags are used to change appearance of text better look and feel than the default text. The formatting tags can make text bold, italic, underlined, etc.

Si.no Tag Usage
1 Bold The HTML Bold Tag defines bold text
2 Italic The HTML Italic Tag changes the text to italic
3 Underline The HTML Underline Tag underlines the text
4 Strong The HTML strong Tag performs similar to Bold but helps in SEO optimizations
5 Emphasize The HTML emphasize tag performs similar to Italic and helps in SEO Optimization
6 Inserted The HTML Inserted tag performs similar to underline and also helps in SEO optimization
7 Mark The HTML Mark tag is used to highlight the text with background color as Yellow
8 Subscript The HTML Subscript tag is used to define subscript text, In small font size
9 Superscript The HTML Superscript tag is used to define superscript text, In small font
10 Small The HTML Small tag is used to make the text to smaller size
11 Big The HTML Big tag is used to make the text to big font size
12 Deleted The HTML Deleted tag is used to display the text in deleted fashion
